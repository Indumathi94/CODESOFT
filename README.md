# CODESOFT
Java Programming Internship Tasks - CodeSoft

This repository contains Java console-based projects developed as part of an internship program.

────────────────────────────

📊 Task 1: Student Grade Calculator  
A program that takes marks for multiple subjects, calculates total marks, average percentage, and assigns a grade based on performance.

────────────────────────────

🏧 Task 2: ATM Interface  
A simple ATM system that allows users to deposit money, withdraw money with validation, and check account balance using object-oriented programming.

────────────────────────────

💱 Task 3: Currency Converter  
A Java application that converts currency from one type to another using exchange rates. It allows user input for amount and selected currencies.

────────────────────────────

🛠 Technologies Used  
☕ Java (Core Java)  
🧠 Object-Oriented Programming (OOP)  
💻 Console-based application
